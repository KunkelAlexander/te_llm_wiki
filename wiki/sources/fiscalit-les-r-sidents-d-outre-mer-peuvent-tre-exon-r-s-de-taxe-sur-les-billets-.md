# Taxation: overseas-territory residents could be exempt from the airline ticket tax, new legal analysis finds

*Original title (French): "Fiscalité : les résidents d'Outre-mer peuvent être exonérés de taxe sur les billets d'avion, selon une nouvelle analyse juridique"*

*Report · France · 2025-10-22*

[Read the original publication](https://www.transportenvironment.org/te-france/articles/fiscalite-les-residents-doutre-mer-peuvent-etre-exoneres-de-taxe-sur-les-billets-davion-selon-une-nouvelle-analyse-juridique)

*(No abstract was published for this document; summary below written from the source text.)*

A legal analysis commissioned by T&E from law firm Baldon Avocats finds that exempting all passengers travelling to and from France's overseas territories from the airline ticket tax (TSBA) would constitute unlawful state aid under EU law. It identifies a narrower option that would hold up legally: exempting only overseas-territory residents themselves on round-trip flights, using the "social character" exemption that EU state-aid rules allow for peripheral-region aviation taxes.
