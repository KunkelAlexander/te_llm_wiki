# Carbon leakage in the aviation sector: Is it a problem and if so what can be done to address it?

*(No abstract was published for this document; summary below written from the source text.)*

A CE Delft study for T&E (March 2025) assessing whether extending the EU ETS to all flights departing EEA airports, not just intra-EEA ones, would cause "carbon leakage" — passengers rerouting via non-EU hubs to avoid the carbon price — and what countermeasures could address that risk.
